File Analyzer

Upload any file to check its type and security status

About This Project

This is a web-based security analysis tool that performs comprehensive file type identification and malware detection. Built with Python Flask and integrated with VirusTotal's API, it provides real-time threat assessment by scanning files against 70+ antivirus engines.

How It Works

🔍
File Type Detection

Analyzes file signatures (magic numbers) to accurately identify file types beyond extensions

🔐
SHA-256 Hashing

Generates cryptographic hash fingerprints for file verification and integrity checking

🛡️
Malware Scanning

Queries VirusTotal API to check files against multiple antivirus engines and threat databases

Tech Stack: Python Flask JavaScript VirusTotal API HTML/CSS
